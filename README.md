############# About the Project   ################
The Gated Community Management System is a Java-based web application built using Servlets, JSP, JDBC, and MySQL. It allows residents to register, log in, submit complaints, update them, and track their status.

Key features include:
- Resident registration and secure login  
- Complaint submission, update, and tracking  
- Admin dashboard for viewing and resolving complaints  
- Optimized JDBC queries and connection pooling  
- Deployment on Apache Tomcat
